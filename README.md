# Front Challenge

## Requirements

- NodeJS: 7.0.0
- Yarn: 0.17.10
- XCode (Mac only, iOS)
- Android SDK (optional)

## Native

```shell
cd native && yarn install
yarn run ios
// or
yarn run android
```

## Web

```shell
cd web && yarn install
yarn start
```

### Build Production

```shell
yarn run build
```
